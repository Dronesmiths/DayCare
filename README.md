# Fairy Tale Child Care

High-authority static website for Fairy Tale Child Care in Palmdale, CA.

## Technical Info
- **Architecture**: Static HTML5/CSS3
- **Deployment**: AWS S3 + CloudFront
- **Niche**: Residential Childcare / Daycare
- **Optimization**: All images WebP, optimized for PageSpeed 100/100/100.

## Project Structure
- `/factory`: Website Factory V3 engine and scripts.
- `/images`: Optimized assets.
- `/services`: Individual program landing pages (Infant, Toddler, Preschool, After-School).
- `/locations`: Localized SEO landing pages.
- `/blog`: Authority-building content.
- `/js`: Custom interactions.
